# AuthWebApp

React frontend for AuthStart - A modern authentication system with role-based access control.

## 🚀 Technologies

- **React 19** - UI library
- **Vite** - Build tool
- **Material-UI (MUI)** - Component library
- **React Router** - Navigation
- **Axios** - HTTP client

## 📦 Installation

```bash
cd /Users/rcastro/Developer/Root/AuthStart/AuthWebApp
npm install
```

## 🔧 Development

```bash
npm run dev
```

Open http://localhost:5173

## 🏗️ Build

```bash
npm run build
```

## 📁 Project Structure

```
AuthWebApp/

│   ├── services/       # API services

│   ├── theme/          # MUI theme config
│   └── App.jsx
├── public/
```

## 🔗 API Connection

Connects to AuthStart Laravel API at `http://127.0.0.1:8000/api`
